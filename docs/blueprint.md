# **App Name**: TaskZenith

## Core Features:

- Secure User Authentication: Enable users to register and log in securely using Firebase Authentication with email/password and Google Sign-In.
- Task Management: Allow users to create, edit, and delete tasks, mark them as complete, set due dates and priorities, and categorize them.
- Real-time Data Sync: Synchronize tasks in real-time across devices using Firebase Firestore.
- Task Prioritization Tool: Leverage generative AI to offer smart suggestions for prioritizing tasks based on their titles and descriptions. This feature will analyze the text content of the tasks to infer importance, deadlines, and relevance, providing dynamic and intelligent recommendations.
- Smart Task Completion Status: A generative AI tool will evaluate a given task description and other properties to assess its likely state of completion given all other tasks for the user and suggest marking certain tasks completed
- Notifications and Reminders: Implement local notifications for task reminders before due dates and daily pending task reminders.
- Search and Filter Tasks: Enable users to search tasks by title and filter them by completion status, priority, and category.
